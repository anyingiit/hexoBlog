Shell(fish)配置docker子命令补全

Posted on 2019-08-21

目标

docker container命令经常使用,但是没有代码补全很难受,每次都要手动输入,所以我们要让docker可以补全子命令container

获取位置

通过$fish_complete_path可以得知fish的补全文件位置

在遍历的所有的目录后发现docker的补全文件在/usr/share/fish/vendor_completions.d/

配置docker.fish文件

complete

complete -c 命令 -s 短选项 -l 长选项 -a 子命令 --description "可选描述"

编辑docker.fish文件并在底部加入

Done!
